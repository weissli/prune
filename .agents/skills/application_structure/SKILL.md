# Application Structure Analysis

Based on the workspace analysis, here is the classification of files distinguishing the core source code (focus of iteration) from distribution and dependencies.

## 📁 Source Files (Focus of Iteration)
These files contain the application logic, components, and styles. Any updates or features will primarily target this directory.

*   **`src/`** - The primary source directory:
    *   **`pages/`**: Contains the main screen views:
        *   `Dashboard.tsx`
        *   `PlantDetail.tsx`
        *   `AddPlant.tsx`
        *   `Welcome.tsx`
        *   `Settings.tsx`
    *   **`components/`**: Reusable UI elements:
        *   `Layout.tsx`
        *   `ui/` (likely containing atomic components like Buttons, Inputs, etc.)
    *   **`services/`**: External API & Data management:
        *   `gemini.ts` (Gemini API integration)
        *   `storage.ts` (Local storage/data handling)
    *   **`hooks/`**: Custom React hooks.
    *   **`lib/`**: Utility functions/libraries.
    *   **`App.tsx`**: Main application router & layout assembly.
    *   **`main.tsx`**: Entry point for React DOM rendering.
    *   **`index.css`**: Global Tailwind/styles.
    *   **`types.ts`**: TypeScript definitions.

*   **`index.html`**: The HTML entry point embedding the React app.

---

## 📦 Distribution & Build Output
These are generated files and should **not** be edited directly during development.

*   **`dist/`**: Contains the production-ready bundle generated by `npm run build`. This is what is actually deployed or served.

---

## 📚 Dependencies
These manage external packages and libraries required by the application.

*   **`node_modules/`**: Downloaded packages from `npm install`. Do not modify.
*   **`package.json`**: Lists dependencies, scripts (e.g., `npm run dev`), and project metadata.
*   **`package-lock.json`**: Hardlocks versions of nested dependencies to ensure consistency.

---

## 🛠️ Configuration
Supporting files that define how the app builds or runs:

*   **`vite.config.ts`**: Configuration for the Vite bundler/server.
*   **`tsconfig.json`**: TypeScript compiler options.
